# ინვენტარის მენეჯმენტის სისტემა - Backend

Node.js/Express ბექენდ API ინვენტარის მენეჯმენტის სისტემისთვის.

## მთავარი ფუნქციონალი

- ინვენტარის CRUD ოპერაციები
- ლოკაციების მართვა
- სტატისტიკის გენერირება
- მონაცემთა ბაზასთან ინტეგრაცია

## ტექნოლოგიები

- Node.js 22 LTS
- Express.js
- Sequelize ORM v6
- PostgreSQL

## ინსტალაცია და გაშვება

1. **რეპოზიტორის კლონირება**:

```bash
git clone https://github.com/Mariamtitvinidze2/inventory-backend.git
cd inventory-backend
npm run start
```
